## 🛠️ How to Use

- Browse into the relevant **Semester** folder.
- Each subject folder contains code examples, notes, and/or assignments.
- Compile C/C++ programs with `gcc` / `g++`:
  ```bash
  gcc hello_world.c -o hello_world && ./hello_world
  ```
- Run Python scripts with:
  ```bash
  python3 script.py
  ```
